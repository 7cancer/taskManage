# taskManage

リポジトリ構成を以下の2系統に整理しました。

- `開発用ディレクトリ/` : ソースコード・設定・ドキュメント・ビルドスクリプトなど開発に必要なファイル
- `配布版ビルド済みディレクトリ/` : 配布用成果物（ZIP など）を配置するディレクトリ

## 開発を始める

```bash
cd 開発用ディレクトリ
npm install
npm run dev -- --host 0.0.0.0
```

## 配布用ZIPを作成する

```bash
cd 開発用ディレクトリ
npm install
npm run build:zip
```

生成物は `../配布版ビルド済みディレクトリ/` に出力されます。
